eslint-plugin-rpgmaker
======================

> [`ESLint`](http://eslint.org) ESlint Plugin for RPG Maker MV and MZ Globals.

## Installation

```sh
$ npm install eslint-plugin-rpgmaker --save-dev
```

## Configuration

```json
{
  "plugins": [
    "rpgmaker"
  ],
  "env": {
    "rpgmaker/mv": true,
    "rpgmaker/mz": true
  }
}
```
